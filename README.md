# 🌟 عالم الأبطال | Hero World Dashboard

قاعدة بيانات وشاشة عرض احترافية (Dashboard) متعددة الألعاب تضم تفاصيل وشخصيات وإحصائيات لأشهر ألعاب الـ Gacha الحالية. مصممة بواجهة مستخدم زجاجية حديثة (Glassmorphism) وتدعم اللغة العربية بالكامل (RTL).

![Hero World Preview](https://i.imgur.com/8xV4LhY.png) <!-- استبدل الرابط بصورة المشروع الحقيقية بعد رفعها -->

## 🎮 الألعاب المدعومة

تم دمج 3 ألعاب حالياً وتعمل البيانات بشكل حيوي بالكامل عبر JSON Data Store:
- **Honkai: Star Rail** (~70+ شخصية)
- **Genshin Impact** (~80+ شخصية)
- **Zenless Zone Zero** (~30+ شخصية)

## ✨ المميزات (Features)
- 🌙 **واجهة داكنة احترافية:** ألوان نيون تتفاعل مع عنصر كل شخصية.
- 🗂️ **فلترة متقدمة لكل لعبة:** الفلاتر تتغير ديناميكياً (مثال: Genshin يعرض أنواع الأسلحة، ZZZ يعرض التخصصات).
- 📱 **متجاوب 100% (Responsive):** تصميم متناسق ومريح يعمل بسلاسة على الهواتف والأجهزة اللوحية (Mobile-First).
- 🔍 **محرك بحث سريع:** للبحث الفوري عن أي شخصية.
- 🇸🇦 **دعم اللغة العربية (RTL):** ترجمة كاملة للعناصر، التخصصات، الإحصائيات، وأنواع الأسلحة.
- 🎨 **تأثيرات بصرية حديثة:** Animations سلسة جداً عبر `Framer Motion` وتأثيرات الـ Shimmer و Pulse Glow.

## 🛠️ التقنيات المستخدمة (Tech Stack)
- **React.js (Vite)**
- **TypeScript**
- **Tailwind CSS** (تصميم الواجهات)
- **Framer Motion** (الأنيميشن والحركة)
- **Lucide React** (الأيقونات)

## 🚀 طريقة التشغيل (How to Run)

1. **تثبيت الحزم (Install Dependencies):**
   ```bash
   npm install
   ```

2. **تشغيل خادم التطوير (Run Development Server):**
   ```bash
   npm run dev
   ```

3. **لبناء نسخة الإنتاج (Build for Production):**
   ```bash
   npm run build
   ```

## 📂 هيكل البيانات (Data Sources)
تعتمد البيانات على استخراج من ملفات الـ API الرسمية ومحفوظة بصيغة JSON خفيفة في مجلد `public/data` للحصول على أداء وسرعة فائقة دون تأخير بالتحميل:
- صور الشخصيات تأتي مباشرة من خوادم **Enka.Network** كـ CDN خارجي.
- بيانات اللغات مدمجة ومترجمة محلياً.
